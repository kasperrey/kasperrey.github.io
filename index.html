Hier staat mijn github repository

Letterkuil is een spel om te leren typen.
Je moet de letters typen voor ze bij het monster aankomen.
Je kan verschillende personen laten spelen.
Er is een winkel waar je met diamanten die je verdient door te spelen, monsters en hoeden voor je monsters kan kopen. Pas op: als je een hoed koopt, dan ben je je vorige hoed kwijt en dat is ook zo bij de monsters.
De letters waarmee je wil spelen kan je aanduiden op een toetsenbord.
Hoe meer letters je aanduid, hoe meer diamanten je verdient.
Als je 10 foute toetsen hebt ingedrukt, dan ga je een level omlaag.
Je ziet de record levels van de andere spelers die op jouw computer spelen.
